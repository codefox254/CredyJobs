# Frontend Documentation

## Overview
React.js frontend for CredyJobs MVP. Public job board and blog, clean and responsive UI.

## Structure
- src/components/ (JobList, JobCard, BlogList, BlogCard, Navbar)
- src/pages/ (Home, Blog, BlogDetail, About)
- App.jsx, index.js

## Setup
1. Install dependencies with npm or yarn
2. Start development server

## Pages
- `/` Home/Jobs
- `/blog` Blog list
- `/blog/:id` Blog detail
- `/about` About

## State Management
- React Query or Context API for fetching jobs/blogs

## Styling
- MUI or Chakra UI recommended
